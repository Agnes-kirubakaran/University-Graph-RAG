# University Graph RAG

A Graph RAG project developed using Python, Neo4j, and OpenRouter.

## Features

- Natural Language Question Answering
- Automatic Cypher Query Generation
- Neo4j Graph Database
- Graph RAG Pipeline
- University Sample Dataset

## Technologies Used

- Python
- Neo4j
- OpenAI SDK
- OpenRouter
- Pandas

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run Neo4j

3. Execute

python graph-rag.py
